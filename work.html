Buscador de Cep

Este é um projeto iniciante de consulta de CEP que utiliza uma API gratuita para fazer a consulta e exibir as informações relacionadas ao CEP, incluindo bairro, logradouro, cidade e UF. O objetivo é criar uma aplicação simples e funcional para ajudar os usuários a encontrar facilmente informações sobre um endereço a partir de seu CEP.
